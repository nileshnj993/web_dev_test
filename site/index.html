Making Pull requests in Github

  • Fork a repository
  • Clone it to your local machine using
    $ git clone "https://github.com/userid/repository"
  • Create a new branch to work on, using $git checkout -b new-branch
  • Stage and Commit local changes using $git add . and
    $ git commit -m "description" respectively.
  • Use $ git status to verify that your changes are committed by Git.
  • You can use the git push command to push the changes to the current branch of your forked repository
    $ git push --set-upstream origin new-branch
